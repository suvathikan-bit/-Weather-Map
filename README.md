How to Run

Clone this repository.

Make sure logo.png is in the project folder.

Install dependencies:

pip install streamlit folium streamlit-folium requests


Run the app:

streamlit run app.py


Click on any location on the map to see weather details.

Use the Settings sidebar to change layers, units, or toggle logo visibility.
